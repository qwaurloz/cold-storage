# Складской учет готовой продукции — АО "Курский хладокомбинат"

## Описание
Программа автоматизирует учет поступающих с конвейера партий мороженого. 
Реализовано 3 базовые функции: добавление партии, просмотр в таблице и удаление.

## Инструкция по запуску
1. Установите pytest для тестов: `py -m pip install pytest`
2. Запустите программу: `py main.py`
3. Запустите тесты: `py -m pytest test_main.py`
